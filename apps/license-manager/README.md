# Raqmi License Manager

Application séparée réservée à l'éditeur de Raqmi System.

## Responsabilités

- Créer les clients / tenants.
- Créer les licences.
- Modifier les modules autorisés.
- Gérer les dates d'expiration.
- Définir les limites : utilisateurs, sites, stockage.
- Suspendre, renouveler ou révoquer une licence.
- Générer des fichiers licence offline signés.
- Suivre les activations serveur.

## Règle de sécurité

Cette application ne doit pas être installée chez le client final. Elle appartient à l'éditeur du logiciel.
