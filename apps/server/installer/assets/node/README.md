# Node.js portable (Windows)

Placez ici `node.exe` (Node.js 20 LTS Windows x64) pour l'inclure dans l'installateur serveur.

Téléchargement : https://nodejs.org/dist/v20.19.0/node-v20.19.0-win-x64.zip

Extraire uniquement `node.exe` dans ce dossier :

```text
installer/assets/node/node.exe
```

Si absent, l'installateur utilisera le Node.js déjà installé sur la machine.
